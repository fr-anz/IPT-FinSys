# Developer Tasks

## Dev 1 - Project Lead / System Integrator

- Manage GitHub repository
- Check project structure
- Integrate all modules
- Final testing

## Dev 2 - Dataset Manager

- Manage raw dataset
- Validate dataset rows and columns
- Prepare data dictionary
- Explain dataset columns

## Dev 3 - Data Cleaning Developer

- Expand `src/data_cleaning.py`
- Handle missing values
- Remove duplicates
- Standardize entries
- Convert data types
- Export cleaned dataset

## Dev 4 - Data Analysis Developer

- Expand `src/analysis.py`
- Compute mean, median, mode, min, max, standard deviation
- Compute category totals
- Compute monthly trends
- Compute budget summaries

## Dev 5 - Visualization Developer

- Expand `src/visualizations.py`
- Create bar graph
- Create line graph
- Create pie chart
- Create histogram or scatter plot

## Dev 6 - Dashboard UI Developer

- Improve `app.py` layout
- Add filters
- Add metric cards
- Organize sections
- Improve dashboard readability

## Dev 7 - Insights Developer

- Expand `src/insights.py`
- Interpret results
- Write findings
- Write recommendations
- Add conclusion text

## Dev 8 - Documentation / Presentation Developer

- Write final paper
- Prepare presentation
- Add screenshots
- Prepare demo script

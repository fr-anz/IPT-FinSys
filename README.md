# Financial Expense Monitoring Dashboard

A Python Streamlit dashboard for the INTE 202 final project. The app loads a raw financial expenses CSV file, cleans and standardizes the data, computes statistical summaries, generates visualizations, and displays insights and recommendations for expense monitoring.

The dashboard now includes filtering, metric cards, analysis tables, charts, cleaned-data preview, and generated findings based on the selected records.

## Setup

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the app:

```bash
streamlit run app.py
```

Open the local Streamlit URL shown in the terminal, usually:

```text
http://localhost:8501
```

## Dataset

Main dataset:

```text
financial_expenses_dataset_raw.csv
```

Expected raw dataset location:

```text
data/raw/financial_expenses_dataset_raw.csv
```

Expected cleaned dataset location:

```text
data/cleaned/financial_expenses_dataset_cleaned.csv
```

Dataset documentation:

```text
docs/dataset_dictionary.md
```

The raw dataset contains financial transaction records with dates, account types, categories, merchants, peso amounts, payment methods, necessity labels, statuses, budget limits, and notes. The cleaned dataset is generated by `src/data_cleaning.py`.

## Features

- Loads the raw financial expenses CSV from `data/raw/`
- Cleans duplicate, missing, inconsistent, and invalid records
- Saves the cleaned dataset to `data/cleaned/`
- Provides sidebar filters for date range, category, payment method, status, and necessity type
- Displays total transactions, total spending, average expense, budget usage, and other key metrics
- Computes mean, median, mode, standard deviation, minimum, maximum, category totals, monthly trends, budget summaries, correlations, and frequency distributions
- Generates bar, line, pie, histogram, and budget comparison charts
- Produces findings, recommendations, and a conclusion from the selected data

## Main Modules

- `src/data_loader.py` handles dataset loading and saving.
- `src/data_cleaning.py` prepares the raw dataset for analysis.
- `src/analysis.py` computes statistics, grouped summaries, correlations, and frequency distributions.
- `src/visualizations.py` creates dashboard charts.
- `src/insights.py` generates findings, recommendations, and conclusion text.
